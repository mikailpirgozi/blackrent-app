🔄 Workflow Použitia

1
Otvorenie Prehľadu

Zamestnanec otvorí sekciu "Dostupnosť vozidiel" v hlavnom menu

2
Výber Zobrazenia

Vyberie si medzi kalendárom, tabuľkou alebo vyhľadávaním podľa dátumu

3
Filtrovanie

Aplikuje potrebné filtre (dátum, typ vozidla, dostupnosť)

4
Výber Vozidla

Klikne na konkrétne vozidlo pre zobrazenie detailu

5
Akcia

Vykoná potrebnú akciu (rezervácia, úprava, vymazanie)

🔧 Technická Implementácia

📋 Databázové Dotazy
┌─────────────────────────────────────────────────────────┐
│  🔍 SQL Dotazy pre Dostupnosť                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│ 1. Všetky vozidlá s dostupnosťou:                      │
│ SELECT v.*,                                             │
│   CASE WHEN r.id IS NULL THEN 'available'              │
│        ELSE 'occupied' END as status                   │
│ FROM vehicles v                                         │
│ LEFT JOIN rentals r ON v.id = r.vehicle_id             │
│   AND CURRENT_DATE BETWEEN r.start_date AND r.end_date │
│                                                         │
│ 2. Dostupné vozidlá v období:                          │
│ SELECT v.* FROM vehicles v                             │
│ WHERE v.id NOT IN (                                     │
│   SELECT DISTINCT vehicle_id FROM rentals              │
│   WHERE (start_date, end_date) OVERLAPS                │
│     ('2024-07-15', '2024-07-17')                       │
│ )                                                       │
│                                                         │
│ 3. Harmonogram vozidla:                                │
│ SELECT r.start_date, r.end_date,                       │
│   c.name as customer_name                              │
│ FROM rentals r                                          │
│ JOIN customers c ON r.customer_id = c.id               │
│ WHERE r.vehicle_id = ?                                 │
│ ORDER BY r.start_date                                   │
└─────────────────────────────────────────────────────────┘

🛠️ Komponenty Systému

  • Frontend: React + Material-UI + FullCalendar
  • Backend: Node.js + Express + PostgreSQL
  • Kalendár: FullCalendar.js s custom rendererom
  • Filtrovanie: Debounced search + React Query
  • Real-time: WebSocket pre live updates
  • Export: jsPDF + xlsx pre PDF/Excel
