📋 Prehľad Rezervačného Systému

Automatizovaný proces rezervácií s inteligentnými notifikáciami

1
Online Rezervácia

Zákazník vyberie vozidlo, dátum a čas cez webové rozhranie

2
Automatické Potvrdenie

Systém odošle email s potvrdením rezervácie

3
Admin Schválenie

Administrátor schváli alebo zamietne rezerváciu

4
Notifikácie

Automatické SMS a email upomienky pred prenájmom

5
Prenájom

Zákazník si vyzdvihne vozidlo s potvrdeným kódom

🛠️ Komponenty Systému

  • Frontend: React + Material-UI
  • Backend: Node.js + Express
  • Databáza: PostgreSQL
  • Email: Nodemailer + SMTP
  • SMS: Twilio API
  • Kalendár: FullCalendar.js
